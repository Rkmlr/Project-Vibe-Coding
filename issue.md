# Perencanaan Pembuatan Frontend Landing Page

Issue ini ditujukan untuk implementasi oleh junior programmer atau AI model, dengan rincian tugas sebagai berikut:

## Panduan Utama
Gunakan pedoman desain dari `@SKILL.md` (`.agents/skills/frontend-design/SKILL.md`) untuk mengoreksi dan mengarahkan pembuatan frontend landing page pada proyek ini.

## Spesifikasi Kebutuhan
1. **Dua Jenis Landing Page**:
   - Landing page khusus untuk **User**.
   - Landing page khusus untuk **Admin**.
2. **Kriteria Desain**:
   - Gunakan desain yang **animatif** dan **elegan**.
   - Pastikan estetika visual mencerminkan standar tinggi sesuai arahan di `SKILL.md`.

## Catatan Tambahan
Pastikan seluruh animasi berjalan secara mulus (smooth micro-animations) dan komponen UI mendukung dark mode elegan (glassmorphism/neon accents) yang sebelumnya sudah dikonsepkan. Evaluasi kembali struktur `page.js` saat ini untuk memisahkan rute user dan admin dengan sempurna.
